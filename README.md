# Convite de Casamento — João e Taís

> Convite digital com design simples em HTML, CSS e JavaScript.

Descrição
- Projeto contendo um convite de casamento digital para João e Taís.
- Estrutura: página principal `index.html`, arquivos estáticos em `assets/`, scripts e estilos.

Como visualizar
1. Abra `index.html` no seu navegador (duplo clique) ou sirva com um servidor simples:

```bash
# Python 3
python -m http.server 8000
# então abra http://localhost:8000/convite-casamento-jorgines/
```

Estrutura do projeto

- `index.html` — página principal do convite.
- `script.js` e `styles.css` — arquivos no diretório raiz.
- `assets/` — scripts e estilos adicionais e possíveis imagens.
